Fitzone Fitness Center



  • HTML
  • CSS
  • JS
  • PHP
  • MySQL
  • Bootstrap




The website is designed to provide users with a comprehensive platform for all their fitness needs. From workout plans to nutrition guides. It includes user signups and schedule as well as admin dashboards hosted on local database server.



    Github     Vercel







It is built using HTML, CSS, and JavaScript, with a focus on responsive design to ensure a seamless experience across devices. The website features a clean and modern layout, making it easy for users to navigate and find the information they need.



Features:

  • Multiple webpages for information.
  • User authentication and signup.
  • Role-based navigation within pages.
  • Role-based dashboards: admin, staff, customer.
  • Dynamic table population.
  • Database CRUD operations in admin dashboard.
  • Responsive design for all devices.
